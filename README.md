# Lark

```
Status: alpha
```

> Get Cloud Project Release Information For DingDing Robot (Toy Project)

## support tracker project

* K8S
* istio

## How to work

### Build Linux

```
./scripts/build.sh
```

### Run your Local cmd

```
bin/lark --token <your_dingding_token>
```

### Docker 

* Build Docker

```
docker build -t lark:0.1 .
```

* Docker from quay

```
docker pull quay.io/klnchu/lark:latest
```
